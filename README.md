# Катапультирующиеся коты (CatApult)

<!-- todo: добавить картинку -->
*Сценарий картинки — [тут](./docs/scenario.md).*

Розы отрастили себе колючки, чтоб их не схрумкали. Улиточки построили себе раковину, чтоб их не раскусили. Некоторые безвредные бабочки косят под ядовитых. «Но это всё — фигня», — заявили соседские коты и эволюционировали в котов катапультирующихся (felis catus catapultis)! Теперь им не страшны нежелательные поглаживания, ведь как ты погладишь того, кто вздымается ввысь, будто ракета?!
